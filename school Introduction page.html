知学塾

個別指導塾のウェブサイト

（TOP・中学生コース・知学塾の特徴・資料請求ページ）

使用したツール・言語

HTML・CSS・Photoshop・Illustrator・AdobeXD・Visual Studio Code

製作期間

26日間

講師によるサポートの充実した個別指導塾を想定。小学1年生から高校3年生をターゲットに、お問い合わせ数や生徒数を増やすことを目的に設定しました。
講師との面談の様子や授業中の写真を起用し、テキストの枠には丸みをつけることで学生や保護者の方に期待と安心感を表現することを意識しました。
「無料体験・資料請求・教室検索」のボタンには立体感を出し、押したくなるデザインを意識しました。
