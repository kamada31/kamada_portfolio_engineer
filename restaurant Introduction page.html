RESTAURANT PREZIOSA

フランス料理店のウェブサイト

（TOP・MENU・STORE・RESERVATIONページ）

使用したツール・言語

HTML・CSS・Photoshop・Illustrator・AdobeXD・Visual Studio Code

製作期間

20日間

世界中から取り寄せた食材を扱うフランス料理店を想定。家族連れや、記念日を迎えた夫婦をターゲットに新規予約を獲得することを目的に設定しました。
店舗の暖かさを伝えるため、手書き風のイラストを配置し、写真はユーザーの食欲をそそるよう彩度を高めに加工しました。
各ページの下部にオンライン予約のバナーを配置し、ナビゲーションメニューの「RESERVATION」はアクセントカラーを用いて強調することで、コンバージョンへ繋げることを意識しました。
